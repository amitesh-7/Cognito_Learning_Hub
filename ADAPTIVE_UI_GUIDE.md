```html
  type="button"
```

- **Space/Enter**: Toggle adaptive mode on/off
- **Tab**: Move to form inputs
- **Disabled inputs**: Skip in tab order when adaptive enabled

```jsx
  useAdaptive && adaptiveInfo && <RecommendationCard data={adaptiveInfo} />;
```

```jsx
const [useAdaptive, setUseAdaptive] = useState(false);
```
